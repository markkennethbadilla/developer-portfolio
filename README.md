# Mark Kenneth Badilla - Developer Portfolio

Personal portfolio website showcasing software engineering projects and technical case studies.

**Live Site:** [mark-kenneth-badilla.vercel.app](https://mark-kenneth-badilla.vercel.app)

## Tech Stack

- **Frontend:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router v6
- **Deployment:** Vercel
- **UI Components:** Custom components with shadcn/ui patterns

## Features

- Responsive design optimized for mobile and desktop
- Technical case study highlighting race condition debugging
- Direct contact form integration
- Performance-optimized with code splitting

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view in browser.

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Route pages (Home, Case Study, Contact)
├── hooks/          # Custom React hooks
└── styles/         # Global styles
```

## Build for Production

```bash
npm run build
npm run preview
```
